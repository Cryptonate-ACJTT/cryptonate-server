# Cryptonate Server

## Commands

```bash
# Install Application
npm install

# Start application
npm start
```

## Technologies

- NPM
- NODE
- TYPESCRIPT (CAN WRITE JS ON IT)
- EXPRESS
- MONGODB
- SWAGGER API - Documenting the API

- routes

- test

- util

- app.ts
